# Application Médiathèque – Gestion de films et séries

## Description du projet

Ce projet est une application console/web en C# qui permet de gérer une médiathèque de films et séries.  
L’objectif principal est de démontrer mes compétences en développement C#, en Entity Framework et en gestion de bases de données relationnelles dans une application structurée et orientée objet.

L’application offre des fonctionnalités complètes de recherche, de filtrage et de suivi des médias, ainsi que la génération de statistiques personnalisées pour chaque utilisateur.

## Fonctionnalités principales

- **Recherche par mots-clés** : trouver facilement un film ou une série dans la médiathèque
- **Filtrage avancé** : trier et filtrer les résultats selon différents critères (genre, année, type de média, etc.)
- **Gestion des états des médias** : marquer un film ou une série comme « vues », « à voir » ou « favoris »
- **Statistiques personnalisées** : consulter des informations comme :
  - Le nombre de films ou séries vus
  - Le genre le plus fréquent
  - La proportion de médias vus / à voir
- **CRUD complet** pour les médias et les utilisateurs (ajout, modification, suppression)
- **Interaction avec la base de données** via Entity Framework (Code First)

## Technologies utilisées

- C#
- Entity Framework Core (Code First)
- SQL Server / base de données relationnelle
- Programmation orientée objet avec classes et interfaces
- Gestion des relations entre entités (utilisateurs, films, séries, genres, etc.)
